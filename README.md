# Crypto Tracker

A modern cryptocurrency tracking application built with React and Vite. This app fetches real-time cryptocurrency data from the CoinGecko API, allowing users to view a list of cryptocurrencies on the home page and detailed information about individual coins.

## Features

- View a list of top cryptocurrencies
- Detailed coin information including price, market cap, and more
- Responsive design for mobile and desktop

## Technologies Used

- React
- Vite
- CoinGecko API
- ESLint for code linting

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
